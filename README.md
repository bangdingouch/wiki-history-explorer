# Wikipedia Timeline Browser

A static GitHub Pages site for exploring Wikipedia articles across revision history.

[Try it out](https://bangdingouch.github.io/wiki-history-browser/?article=Wikipedia&rev=1355374251)

## Features

- Revisions timeline
- Inline added/removed diffs
- Keyboard navigation            (use arrow keys)
- Revision metadata
- Article length chart

## Publish

Upload `index.html`, `app.js`, `styles.css`, and `.nojekyll` to your repo, then enable GitHub Pages from the `main` branch root.
